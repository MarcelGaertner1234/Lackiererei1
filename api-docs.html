Fahrzeugannahmen abrufen

GET /intakes

Ruft alle Fahrzeugannahmen ab. UnterstÃ¼tzt Pagination und Filter.

Query Parameter:
  • limit - Anzahl der Ergebnisse (max 100, default 20)
  • offset - Skip n Ergebnisse
  • status - Filter nach Status (open, completed, archived)
  • from - Filter nach Datum (ISO 8601)
  • to - Filter bis Datum (ISO 8601)

GET /intakes?limit=10&status=completed

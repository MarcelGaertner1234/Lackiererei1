Erste Schritte Bedienung Support

Erste Schritte

1. Registrieren Sie sich kostenlos auf unserer Website
2. Bestätigen Sie Ihre E-Mail-Adresse
3. Loggen Sie sich ein und folgen Sie dem Onboarding-Assistenten
4. Importieren Sie optional Ihre bestehenden Kundendaten
5. Fertig! Sie können sofort mit der Fahrzeugannahme starten.

VehicleFlow läuft auf allen modernen Geräten: iPad (empfohlen), Android-Tablets, Desktop-PCs (Windows/Mac) und Smartphones. Mindestanforderungen: iOS 12+, Android 8+, Chrome 90+, Safari 13+.

Bedienung

Klicken Sie auf "+ Neue Annahme" → Wählen Sie den Service-Typ → Füllen Sie die Fahrzeugdaten aus → Machen Sie Fotos der Schäden → Lassen Sie den Kunden unterschreiben → Fertig! Das PDF wird automatisch erstellt.

Gehen Sie zu Einstellungen → Team → "+ Mitglied hinzufügen" → E-Mail-Adresse eingeben → Rolle zuweisen (Admin/Mitarbeiter) → Einladung senden. Der Kollege erhält eine E-Mail mit einem Einladungslink.

Einstellungen → Daten & Export → Wählen Sie das Format (Excel/CSV/PDF) → Wählen Sie den Zeitraum → "Exportieren" klicken. Der Download startet automatisch.

Mitarbeiter Migration

Von localStorage zu Firebase Authentication

Wichtige Hinweise

  • Backup: Alle localStorage Daten werden als Backup in Firestore gespeichert
  • E-Mail erforderlich: Jeder Mitarbeiter braucht eine gültige E-Mail-Adresse
  • Temporäre Passwörter: Nach Migration erhalten Mitarbeiter temporäre Passwörter
  • Internet erforderlich: Migration funktioniert nur mit aktiver Internetverbindung
1

localStorage auslesen

Alle Mitarbeiter aus localStorage werden geladen und angezeigt.

2

E-Mails eingeben

Für jeden Mitarbeiter eine gültige E-Mail-Adresse eingeben.

3

Migration starten

Firebase Auth Accounts werden erstellt und Daten kopiert.
